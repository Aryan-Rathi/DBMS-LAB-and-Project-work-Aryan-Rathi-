Experiment 5: Aggregate and String Functions
Problem Statement
The objective of this experiment is to use built-in SQL functions to perform mathematical summaries on data and manipulate string formats for reporting.

Queries to Implement:
1. Display the total number of employees working in the company.

2. Display the total salary being paid to all employees.

3. Display the maximum salary from the employee table.

4. Display the minimum salary from the employee table.

5. Display the average salary from the employee table.

6. Display the maximum salary being paid to a Clerk.

7. Display the maximum salary being paid in Dept No 20.

8. Display the minimum salary paid to any Salesman.

9. Display the average salary drawn by Managers.

10. Display the total salary drawn by Analysts working in Dept No 40.

11. Display the names of the employees in Uppercase.

12. Display the names of the employees in Lowercase.

13. Display the names of the employees in Proper case (Initcap).

14. Display the length of your name using an appropriate function.

15. Display the length of all the employee names.
